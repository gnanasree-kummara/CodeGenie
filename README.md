# 🚀 Code Genie – AI Code Generator & Explainer

## 📌 Overview

Code Genie is a web-based AI application that generates code from natural language input and provides simple explanations. It supports multiple programming languages and helps users understand coding concepts easily.

## ✨ Features

* AI-based code generation
* Code explanation in simple English
* Multi-language support (Python, Java, C, C++, JavaScript)
* User authentication system
* Admin dashboard

## 🛠️ Tech Stack

* Frontend: HTML, CSS
* Backend: Python (Flask)
* Database: SQLite
* API: Hugging Face
